# 凡人修仙传·命运模拟 v2 设计文档

**日期：** 2026-04-16
**版本：** v2.0
**范围：** 在 v1.0 基础上扩展分支结构（二层分支）和结局数量（7→16个），提升探索驱动的留存性

---

## 1. 设计目标

**核心目标：探索驱动的留存性。**

用户能感知到"还有没解锁的结局"，并有动力反复游玩。每次选择路径不同，经历的题目不同，获得的结局不同。16个结局需要多次游玩才能全部解锁。

---

## 2. 分支结构（二层分支）

### 2.1 整体流程

```
公共主干 Q1–Q5（5题）
    ↓ Q5 触发一级分支
    ├── 分支A：入世路线（加入门派）
    │       QA1–QA3 共干（3题）
    │       ↓ QAX 分叉题（触发二级分支）
    │       ├── A1：征战路线 QAA1–QAA4（4题）
    │       └── A2：权谋路线 QAB1–QAB4（4题）
    │
    └── 分支B：出世路线（独自游历）
            QB1–QB3 共干（3题）
            ↓ QBX 分叉题（触发二级分支）
            ├── B1：求道路线 QBA1–QBA4（4题）
            └── B2：江湖路线 QBB1–QBB4（4题）

汇合收尾 QF1–QF3（3题）
    ↓
结算 → 结果页
```

### 2.2 每次游玩题数

每次玩家经历：5（主干）+ 3（一级共干）+ 1（分叉）+ 4（二级支线）+ 3（收尾）= **16题**，约5分钟。

### 2.3 路径与结局池

| 路径 | 描述 | 可触发结局 |
|------|------|-----------|
| A1 征战 | 门派战场，以力为尊 | 飞升、斗法陨落、魔修、制霸一方、守护牺牲 |
| A2 权谋 | 门派内斗，深谋远虑 | 飞升、心魔陨落、背刺陨落、双修伴侣、散修宗师 |
| B1 求道 | 独游悟道，参透天地 | 化为天地、隐世、涅槃重生、散修宗师、天降机缘 |
| B2 江湖 | 漂泊江湖，阅尽人情 | 回归凡尘、背刺陨落、守护牺牲、被仙人接引、自然坐化 |

## 4. 新增题目详细设计

### 4.1 分支A 分叉题（QAX）

**【QAX · 普通选项 · 触发二级分支】**
> 在门派中历练数年，你逐渐找到了自己的位置。师兄弟们都说，你是那种——

- A. 冲锋在前的人，战场上才是你的归宿 → **A1征战** · 勇气+10 野心+5
- B. 深谋远虑的人，棋局之中才见真章 → **A2权谋** · 智慧+10 JP→J

### 4.2 A1 征战路线（QAA1–QAA4）

**【QAA1 · 图片场景选择】**
> 一场大战在即，你主动请缨担任——

- A. 先锋突破，第一个冲入敌阵（勇气+15 野心+5）
- B. 断后殿军，保证同伴安全撤退（情义+15 勇气+5）
- C. 奇兵绕后，出其不意断敌退路（智慧+10 勇气+10）

**【QAA2 · 滑动条】**
> 战场上你的杀戮欲——
> 极强，战斗是你最真实的状态 ←————→ 克制，战斗只是手段

- 偏左（勇气+15 情义-15）
- 居中（勇气+5 智慧+5）
- 偏右（智慧+10 情义+5）

**【QAA3 · 普通选项】**
> 你的至交在战场上被敌方俘虏，敌方开出条件：用你的修为换他自由。你——

- A. 答应，修为可以再练，朋友只有一个（情义+25 野心-10）
- B. 拒绝，以武力强行救人（勇气+20 情义+10）
- C. 假意答应，暗中布局（智慧+15 勇气+10）

**【QAA4 · 普通选项】**
> 你在战场上声名大噪，有人私下找你，说可以帮你走上更高的位置，但需要出卖门派的一个秘密。你——

- A. 断然拒绝，当场击杀来人（勇气+10 情义+10）
- B. 假装答应，暗中报告掌门（智慧+15 情义+5）
- C. 动摇了，最终还是拒绝（智慧+5 野心-5）

## 5. 新增结局文案

### 魔修（moxiu）
**故事：** 战场上的杀戮让你尝到了一种奇异的快感。那一天，你在修炼时走火入魔，灵台中涌出一股黑色的气流。你没有抗拒——那感觉太熟悉了，就像战场上的你。从此，你踏上了魔道，再也回不了头。
**关键词：** 走火入魔、快感成瘾、不归路、魔道修士
**性格简述：** 你的欲望和力量都超过了你的自制力。魔道不是堕落，对你而言，那才是真实的自己。

### 制霸一方（bawang）
**故事：** 你的名字让人闻风丧胆。你没有飞升，也不想飞升——这片土地上的一切都在你的掌控之中，这就够了。你建立了自己的王国，以绝对的武力维系秩序。只是，众矢之的的滋味，并不好受。
**关键词：** 铁腕统治、孤家寡人、众矢之的、强权为尊
**性格简述：** 你用力量换来了掌控，却也用力量换走了温度。站在最高处，你才发现四面都是敌人。

### 心魔陨落（xinmo）
**故事：** 渡劫那一刻，心魔现身了。它的面孔是你自己——那个被欲望和野心填满、早已忘记初心的你。你与它缠斗了七天七夜，最终，你分不清哪个才是真正的自己。劫云散去，什么都没有留下。
**关键词：** 执念成魔、自我迷失、心魔噬心、欲望代价
**性格简述：** 野心是你最大的驱动力，也是你最深的陷阱。当你忘记了为何而修，修仙本身就成了你的劫。

### 背刺陨落（beici）
**故事：** 那一剑来自你最信任的人。你的情义让你放松了警惕，而你的智慧不够，没能看穿那张温情的脸背后藏着的刀。你倒下的那一刻，心里只有一个念头：原来如此。
**关键词：** 信任代价、防人之心、温情背后、死于背叛
**性格简述：** 你给出了全部的信任，却没有留下足够的清醒。这不是你的错，但你为此付出了生命。

### 双修伴侣（shuangxiu）
**故事：** 有人与你并肩走过了最艰难的岁月。渡劫那天，你们同时踏入劫云，彼此以神魂相护。劫云散去，两道身影冲破苍穹。修仙路漫漫，但你不是一个人。
**关键词：** 并肩同行、神魂相护、共渡飞升、情义与力量
**性格简述：** 你证明了，情义和力量不是对立的。最深的羁绊，有时候反而是最强的修炼。

### 散修宗师（zongshi）
**故事：** 你从未加入任何宗门，却桃李满天下。你将毕生所悟整理成册，收下了一批又一批弟子。你没有飞升，但你的道，在一代又一代人身上延续。也许这，才是另一种永生。
**关键词：** 桃李天下、自创一派、道统传承、另一种永生
**性格简述：** 你的智慧让你看到了比长生更深远的东西——传承。你选择了留下，而不是飞升，因为你知道什么更重要。

### 化为天地（tiandi）
**故事：** 寿元将尽那日，你没有遗憾。你盘膝而坐，将神魂缓缓向外扩散，融入山川，融入云雾，融入每一缕风。你消失了，又无处不在。后来的修士说，这片山中有一种奇异的灵气，让人心静。那是你。
**关键词：** 神魂化虚、天地同寿、无处不在、另一种永存
**性格简述：** 你的智慧和情义让你选择了最罕见的归宿——不飞升，不坐化，而是融入这片你守护过的天地。

### 涅槃重生（niepan）—— 彩蛋
**故事：** 渡劫失败，神魂本该消散。但那一刻，你的执念太深——不是对长生的执念，而是对未竟之事的执念。神魂在虚空中漂荡了不知多久，最终凝聚成一个婴儿，降生在某个普通农家。这一世，你记得一切。
**关键词：** 神魂不散、执念重生、记忆留存、重头再来
**性格简述：** 你死过一次，却又回来了。带着上一世的记忆，这一次，你会做出不同的选择吗？

### 回归凡尘（fanchen）
**故事：** 修为在某一天停滞了，再也无法寸进。你在一个小镇住下来，开了一间药铺。街坊邻居不知道你曾是修士，只知道这个老大夫医术高明，脾气温和。你也渐渐忘了，曾经那么渴望长生。
**关键词：** 修为停滞、归隐凡尘、平凡温暖、放下执念
**性格简述：** 修仙是你走过的路，但凡尘才是你最终的归宿。也许从一开始，你就不是那种注定要飞升的人。

### 被仙人接引（xianyou）—— 超稀有彩蛋（0.3%）
**故事：** 那一天，你正在江湖某处的破庙里避雨，百无聊赖地打坐。忽然，庙外金光大盛，一位白衣老者从天而降，看了你一眼，说了一句话：「与其在这泥泞里挣扎，不如随我去。」你甚至来不及问他是谁，就已经被一股力量托起，穿破云层。破庙里只留下你的衣物，和一个目瞪口呆的流浪汉。
**关键词：** 天外来客、无需渡劫、被动飞升、万古奇遇
**性格简述：** 你没有刻意追求飞升，飞升却主动找上了你。有些人，天生就是被命运偏爱的。

```json
  "options": [
    { "text": "...", "scores": {...}, "branch2": "A1" },

  ]
}
```

# 凡人修仙传·人界篇 MBTI 人物匹配测试 设计文档

**日期：** 2026-04-14  
**范围：** 仅 MBTI 人物匹配测试模块（网站其他模块后续迭代）

---

## 1. 项目概述

一个以《凡人修仙传》人界篇为主题的网站，首期只实现 MBTI 人物匹配测试功能。用户通过回答修仙世界情景题，系统在内部用 MBTI 四维度计算，最终匹配出最接近的人界人物，结果页支持社交分享。

**目标：** 面向凡人修仙传粉丝群体，内容有趣、易传播。

## 4. 功能设计

### 4.1 测试流程

```
首页 → 点击"开始测试" → 逐题作答（情景选择题）→ 计算结果 → 结果页（可分享）
```

### 4.2 题目设计

- **题目数量：** 12 题
- **题目形式：** 修仙世界情景选择题，每题 2 个选项（A/B）
- **内部维度：** 每道题对应 MBTI 四维度之一（E/I、S/N、T/F、J/P），每个维度 3 题
- **用户感知：** 只看到修仙情景，不感知 MBTI 维度

**示例题目：**
> 你在七玄门修炼多年，师父给了你两个选择：
> - A. 留在门中继续深造，稳扎稳打
> - B. 独自出门历练，见识更广阔的世界
>
> （内部：对应 J/P 维度，A → J，B → P）

### 4.3 结果计算

每个维度统计 A/B 选择数量，多数方向即为该维度结果，组合出 16 种 MBTI 类型，每种类型对应一个人界人物。

### 4.5 结果页内容

- 人物名称 + 人物简介（2-3句）
- MBTI 类型标签
- 性格特点（3-4个标签）
- 人物在书中的经典语录或事迹（1条）
- **分享按钮：** 复制链接 / 分享到微博 / 下载结果卡片图（可选）
- **重新测试按钮**

## 5. 页面结构

```
/                    ← 首页（Hero + 测试入口）
/quiz                ← 测试页（逐题作答）
/result/[mbti]       ← 结果页，如 /result/intj → 韩立
```

### 5.1 首页

- 顶部导航：Logo + 占位导航项（人物百科、地图势力、剧情时间线，暂时 disabled）
- Hero 区域（左右分栏）：
  - 左：网站标题"凡人修仙传·人界篇"+ 简介 + "探索人界"按钮（暂时跳转测试）
  - 右：突出显示"你是哪位修仙人物？"+ "立即测试"按钮
- 下方三个功能预览卡片（暂时显示"敬请期待"）
- Footer：版权声明

### 5.2 测试页 `/quiz`

- 进度条（第 X / 12 题）
- 情景题文字
- 两个选项卡片（点击选择，自动进入下一题）
- 不允许回退（保持测试流畅性）

### 5.3 结果页 `/result/[mbti]`

- 人物名称大字展示
- MBTI 类型 + 性格描述
- 分享区域
- 重新测试入口

## 6. 数据文件结构

```
content/
  quiz/
    questions.json     ← 12道题目，每题含文字、选项A/B、对应维度和方向
    characters.json    ← 16个人物，含 mbti、名字、简介、特点、语录
```

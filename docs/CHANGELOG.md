# 凡人修仙传·命运模拟 版本迭代记录

> 记录从 MBTI 测试到命运模拟游戏的完整演进历程

---

## v1.0 — 命运模拟基础版
**完成时间：** 2026-04-15

### 核心功能
将原有 MBTI 性格测试升级为"修仙命运模拟"游戏。用户完成20道题后，不再只是得到性格类型，而是获得完整的修仙人生结局。

### 主要内容

**计分体系（双轨并行）：**
- 命运维度：勇气、智慧、情义、野心（各 0-100）
- MBTI 维度：EI / SN / TF / JP（投票取多数，平票取 I/N/F/P）

**题目系统（20题）：**
- 公共主干 Q1-Q5（5题）
- 分支A：勇猛路线 QA1-QA6（6题）
- 分支B：谋略路线 QB1-QB6（6题）
- 汇合收尾 QF1-QF3（3题）
- 题型：普通选项、滑动条（三段计分）、排序拖拽、图片场景、限时题（5秒倒计时）

**5个境界：** 炼气期 → 筑基期 → 结丹期 → 元婴期 → 化神期
- 由`勇气 + 野心`之和决定

**7种结局：** 飞升、境界突破失败陨落、守护他人牺牲、隐世、斗法陨落、自然坐化、天降机缘（彩蛋1%）

**结果页（三层布局）：**
- 第一层：境界 + 寿元 + 结局故事
- 第二层：四维度雷达图 + 性格关键词
- 第三层：最匹配的凡人世界角色（基于 MBTI）

## v2.0 — 二层分支 + 16个结局
**完成时间：** 2026-04-16

### 核心目标
提升探索驱动的留存性。用户需要多次游玩才能解锁全部16个结局。

### 主要变更

**分支结构扩展（一层 → 二层）：**
```
Q5 → 分支A → QAX分叉 → A1征战路线（QAA1-4）
                      → A2权谋路线（QAB1-4）
     分支B → QBX分叉 → B1求道路线（QBA1-4）
                      → B2江湖路线（QBB1-4）
```
每次游玩经历16题，4条路径各有独特的题目和结局池。

**结局扩展（7 → 16个）：**

| 新增结局 | 路径 | 触发条件 |
|---------|------|---------|
| 魔修 | A1 | 勇气≥70 且 情义＜30 |
| 制霸一方 | A1 | 勇气≥80 且 智慧＜30 且 情义＜30 |
| 心魔陨落 | A2 | 野心≥80 且 情义＜25 |
| 背刺陨落 | A2/B2 | 情义≥70 且 智慧＜35 |
| 双修伴侣 | A2 | 情义≥80 且 勇气≥60 |
| 散修宗师 | A2/B1 | 智慧≥70 且 野心＜40 且 勇气≥50 |

## v3.0 — 动态题目系统
**完成时间：** 2026-04-16

### 核心目标
让每次游玩体验真正有差异——不只是走不同路径，而是看到不同的题目内容。

### 主要变更

**动态题目架构：**
- 32道内容题改为 `versions` 结构，每题2-3个版本
- 6道固定题（Q5/QAX/QBX/QF1-3）保持不变
- 版本选择逻辑：`prevAnswer` 精确匹配 > `score` 阈值 > `default`

**版本触发条件：**
- `prevAnswer`：上一题选了某个特定选项（最高优先级）
- `score`：累积分数达到阈值（如 courage≥50 显示"你已按捺不住"版本）
- `default`：兜底版本

**示例：** Q2（选择修炼方向）
- Q1 选"云雾缭绕的深山洞府" → "道人将你带入深山，问你想走哪条路"
- Q1 选"热闹繁华的修仙坊市" → "在坊市见识了各路修士的风采"
- Q1 选"宁静清幽的竹林小筑" → "在竹林中静心数日，你领悟到自己的方向"
- 其他 → 原版题干

**新增 `resolveQuestion` 函数**，在渲染每道题前动态选出当前版本。

## v4.0 — 向道之心 + 机遇值 + 冲关概率
**完成时间：** 2026-04-17

### 核心目标
让境界不再是"分数够了就到达"，而是需要真正的修道意志和天赋机遇才能突破。

### 主要变更

**新增两个隐藏维度：**

| 维度 | 字段 | 说明 |
|------|------|------|
| 向道之心 | `perseverance` | 题目累积，0-100，雷达图可见 |
| 机遇值 | `fortune` | 游玩开始时随机生成1-100，全程不变，用户不可见 |

**境界计算重构（逐级冲关）：**
- 触发条件改为 `勇气 + 向道之心` 之和（替换原来的勇气+野心）
- 每级冲关有独立成功率，失败直接陨落（tupo）

| 冲关 | 基础成功率 | perseverance加成 | fortune加成 |
|------|-----------|-----------------|------------|
| 炼气→筑基 | 70% | +p×0.2% | +f×0.1% |
| 筑基→结丹 | 35% | +p×0.3% | +f×0.15% |
| 结丹→元婴 | 15% | +p×0.4% | +f×0.2% |
| 元婴→化神 | 10% | +p×0.5% | +f×0.25% |

**飞升条件更新：** `化神期 + 向道之心 ≥ 80`（原为野心≥80）

**被仙人接引概率：** `fortune / 100 × 3%`（B2路径，机遇值越高越容易触发）

**雷达图升级为5维度：** 勇气 / 悟性 / 情义 / 野心 / 向道之心

**向道之心加分场景（题目中）：**
- 放弃世俗地位追求更高境界
- 选择孤独闭关而非繁华
- 遇到挫折不退缩
- 面对死亡仍想突破

## v5.0 — 全站视觉升级（夜墨深色主题）

- **翠玉绿（主色）：** jade-400(`#4ade9a`) 为核心交互色

- **雾白（正文）：** mist-100(`#e8f0ec`)

### 主要改动
- **全局粒子背景：** `ParticleBackground` 组件，80粒子（88%翠绿+12%金色），全站共享
- **三层渐变光晕背景：** 左上翠绿、右下金色、中心深翠绿
- **通用卡片样式：** `card-glow`（渐变发光描边）、`card-death`（红色裂纹纹理）
- **按钮系统：** `btn-primary`（翠绿发光）、`btn-secondary`（透明边框）、`btn-death`（红色裂纹）

## 后续规划

### 近期小迭代
- **结局图鉴：** localStorage 记录已解锁结局，展示收集进度
- **分享图片生成：** 结果页生成可分享的图片卡片
- **冲关动效：** 渡劫时有专属演出（闪光、倒计时、成功/失败特效）

### 长期方向：从测试到游戏

**核心转变：** 将 `/destiny` 从"命运测试"演变为真正的**交互式修仙模拟游戏**

**核心循环：**
```
进入境界阶段 → 自动积累资源 + 随机事件处理
→ 寿命倒计时施压 → 玩家主动决定冲关
→ 成功进入下一境界（正反馈） / 失败概率陨落（游戏结束）
```

**寿命系统：**
- 每个境界有正常寿命上限 + 极限寿命上限
- 天材地宝可延寿，不能超过极限上限
- 寿命耗尽 = 自然坐化，游戏结束
- 突破境界是增加寿命上限的唯一主要手段

**资源系统：**
- 自动积累（随时间/修炼轮数增长）
- 事件掉落（丹药、灵石、阵法等），不同选择给不同奖励/惩罚
- 资源积累影响冲关成功率
- 典型事件：加入宗门、奇遇、劫难、机缘

**节奏设计（目标15分钟一局）：**
- 阶段制，每个境界节奏相似，都有足够内容和重量感
- 玩家在每个境界感受到"在这里生活、修炼、做选择"
- 每个境界约 3-5 个事件，冲关难度随境界递增
- 正确决策 + 资源积累 → 达到更高境界 → 成就感正反馈

**MBTI：** 后续逐步淡化，游戏机制优先于性格测试

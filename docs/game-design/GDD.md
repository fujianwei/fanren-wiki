# 凡人修仙传·命运模拟 游戏设计文档（GDD）

> 版本：v1.0 | 日期：2026-04-18

---

## 一、游戏概述

### 核心定位
交互式修仙模拟游戏。玩家经历轮回转世获得灵根，踏入修仙之路，通过事件决策、资源积累、冲关突破，最终尝试飞升灵界。

### 核心体验
- **不是测试**，而是一次真实的修仙人生体验
- **正反馈驱动**：正确决策 + 资源积累 → 突破更高境界 → 成就感
- **寿命即压力**：时间有限，每个选择都有代价
- **目标游玩时长**：15分钟一局

### 核心循环
```
进入境界阶段
→ 自动积累资源 + 随机事件处理（主动/被动）
→ 寿命倒计时施压
→ 玩家主动决定冲关
→ 成功：进入下一境界（正反馈）
→ 失败：概率陨落 / 维持境界继续积累
```

## 二、开局流程

```
轮回入场 → 随机循环直到 random > 95 → 获得灵根 → 直接进入炼气期
```

灵根品质由机遇值（fortune）决定，无需额外条件直接开始游戏。

### 灵根品质
| 灵根 | 机遇值范围 | 修炼速度倍率 |
|------|----------|-----------|
| 天灵根 | 91-100 | ×2.0 |
| 双灵根 | 81-90 | ×1.4 |
| 三灵根 | 51-80 | ×1.0（基准） |
| 伪灵根 | 1-50 | ×0.5 |

**核心设计逻辑**：灵根越好 → 修炼越快 → 满修为消耗事件越少 → 剩余寿命越多 → 可做更多主动事件 → 资源越丰厚 → 冲关成功率越高。天灵根与伪灵根差距4倍，伪灵根玩家必须逼近寿命极限才能完成突破。

## 三、境界系统

### 五大境界
| 境界 | 正常寿命上限 | 极限寿命上限 |
|------|------------|------------|
| 炼气期 | 100年 | 150年 |
| 筑基期 | 200年 | 300年 |
| 结丹期 | 500年 | 800年 |
| 元婴期 | 1000年 | 1500年 |
| 化神期 | 2000年 | 3000年 |

魔道路线寿命上限整体-15%。天材地宝可延寿，不能超过极限上限。

### 修为层次（每个境界内）
| 修为进度 | 阶段 | 说明 |
|---------|------|------|
| 1-30% | 初期 | 自然积累，无需操作 |
| 31-70% | 中期 | 自然积累，无需操作 |
| 71-89% | 后期 | 自然积累，无需操作 |
| 90-99% | 后期巅峰 | 解锁冲关资格 |
| 100% | 圆满 | 冲关成功率最高 |

小境界（初/中/后期）自然突破，无需冲关。

### 每事件修为增加（基准值，三灵根）
| 事件类型 | 修为增加 |
|---------|---------|
| 普通事件 | +10点 |
| 闭关（短期） | +5点 |
| 闭关（中期） | +10点 |
| 闭关（长期） | +18点 |

**所有修为增加 × 灵根倍率**（天灵根×2，双灵根×1.4，三灵根×1，伪灵根×0.5）

## 六、事件系统

### 事件分类

**被动事件**（系统随机推送，无法拒绝遭遇）：
- 路遇受伤散修
- 宗门任务来了
- 仇人找上门
- 奇遇/劫难

**主动事件**（玩家主动触发，可选择做或不做）：
- 闭关修炼
- 进入副本
- 外出历练
- 去市场买卖
- 主动求战

### 事件池
- 每个境界独立事件池，20-30个事件
- 随机抽取，可以重复出现
- 5个境界总计100-150个事件

| 类型 | 占比 | 每境界数量 |
|------|------|----------|
| 普通事件 | 50% | 10-15个 |
| 因果事件 | 20% | 4-6个 |
| 副本 | 15% | 3-5个 |
| 战斗 | 15% | 3-5个 |

### 事件结构示例
```
【事件】山中偶遇一位受伤的散修
他身上有一颗品质不错的筑基丹，但神情警惕。

A. 出手救治，以诚相待
B. 趁机索要丹药，以武力威胁
C. 视而不见，继续赶路
D. 【需要：灵石×50】出价购买
```

### 副本机制
- 副本是主动事件的一种，高风险高收益
- 由多个连续事件组成，可中途撤退
- 撤退收益按进度计算：

| 完成进度 | 撤退收益 |
|---------|---------|
| 完成1/3 | 20%奖励 |
| 完成1/2 | 40%奖励 |
| 完成2/3 | 60%奖励 |
| 全部完成 | 100%奖励 |

## 七、因果链系统

- 正负因果数量对等，负面不低于正面
- 触发：必然触发（重要因果）+ 概率触发（小恩小惠）
- 跨度：同境界短线 + 跨境界长线
- **暗示机制**：角色台词提前告知（"他日必来报恩"）

**典型正面因果：**
- 救助受伤散修 → 后续带来机缘
- 帮助筑基修士（实为被封印化神）→ 冲击化神时获得关键丹药

**典型负面因果：**
- 修炼魔功 → 冲关时心魔干扰-20%
- 背叛宗门 → 被通缉，买不到丹药
- 强抢机缘 → 结下仇家，某境界遭遇报复
- 滥杀无辜 → 杀戮之气积累，寿命悄悄损耗
- 贪图近道服丹 → 根基不稳，下次冲关成功率下降
- 出卖同门 → 关键时刻被坑

## 八、冲关系统

### 冲关基础成功率（修为100%时）
| 冲关 | 基础成功率 | 失败陨落概率 | 消耗寿命 |
|------|----------|-----------|---------|
| 炼气→筑基 | 70% | 极低 | 8年 |
| 筑基→结丹 | 50% | 极低 | 25年 |
| 结丹→元婴 | 20% | **90%** | 80年 |
| 元婴→化神 | 35% | 50% | 200年 |

修为与成功率关系：
- 90%修为：基础×0.5
- 95%修为：基础×0.7
- 99%修为：基础×0.85
- 100%修为：基础×1.0

### 冲关成功率加成
| 来源 | 加成 |
|------|------|
| 筑基丹 | +15% |
| 降尘丹 | +15% |
| 补天丹 | +15% |
| 绛云丹 | +15% |
| 聚灵阵 | +8% |
| 护体阵 | +12% |
| 天罡阵 | +20% |
| 灵石（每500） | +3%，上限+9% |
| 根基稳固 | +10% |

| 经验值（每次失败） | +3%，上限+15% |

### 结丹→元婴最优情况
```
20%（基础）+ 15%（补天丹）+ 20%（天罡阵）+ 9%（灵石）+ 10%（根基）= 74%
心魔关另算：正面情绪积累 + 九曲灵参丸+20%
```

### 心性/情绪系统（仅结丹→元婴）
**冲关公式**：总成功率 = 正常冲关概率 × 战胜心魔概率

战胜心魔概率由正负情绪差值决定：
| 差值（正-负） | 战胜心魔概率 |
|------------|------------|
| ≥50 | 80% |
| 20-49 | 60% |
| 0-19 | 40% |
| -20到-1 | 20% |
| ≤-21 | 5% |

| 行为 | 类型 | 加值 |

| 屡败屡战不放弃 | 坚韧 | +10 |
| 放弃资源选择大义 | 淡泊 | +8 |
| 行善积德 | 道义 | +6 |

**负面情绪积累：**
| 行为 | 类型 | 加值 |
|------|------|------|
| 杀戮/背叛 | 戾气 | +10 |
| 走捷径/强抢 | 贪念 | +8 |
| 过度追求某物 | 执念 | +6 |
| 冲关失败 | 恐惧 | +8 |
| 临阵逃跑 | 恐惧 | +5 |

**消除负面情绪：**
- 闭关静心：消耗寿命+灵石

## 十、市场系统

- 固定存在，随时访问
- 每次显示3个随机商品
- 刷新免费无限次
- 购买只消耗灵石，固定价格

### 灵石获取途径
事件奖励、副本掉落、战斗胜利、宗门任务、卖道具、采集、散修劫道（负面因果）

## 十一、战斗系统

### 战斗场景
- **仇人战斗**：被动，因果触发，躲不掉
- **争夺资源战斗**：半主动，玩家决定要不要抢
- **历练主动求战**：纯主动，为修为/资源主动发起

### 战斗流程
1. 显示对方境界和实力评估（强于你/相当/弱于你）
2. 玩家决定战或逃
3. 3回合关键决策，每回合选项可消耗资源
4. 综合计算胜负

### 胜负计算
| 3回合总分 | 判定 |
|---------|------|
| +3或+2 | 完胜，对方重伤/死亡 |
| +1 | 小胜，对方轻伤，玩家无伤 |
| 0 | 平局，双方轻伤 |
| -1 | 小败，玩家轻伤 |
| -2 | 败，玩家重伤 |
| -3 | 惨败，玩家濒死 |

- 境界差距修正：对方每高一级-1，每低一级+1
- 散修天然每回合+1胜负倾向

### 逃跑机制
- 损失一定灵石
- 隐性积累负面情绪（恐惧）
- 第三次逃跑触发提示

### 输赢后果
| 场景 | 赢了 | 输了 |
|------|------|------|
| 仇人战斗 | 消除负面因果，可能获得资源 | 受伤/陨落 |
| 争夺资源 | 获得资源，积累戾气 | 受伤，没抢到 |
| 历练求战 | 修为+，可能获得道具 | 受伤 |

### 仇人系统
- 一次性：结仇 → 仇人池 → 后续概率遭遇 → 战斗解决后移除
- 仇人境界：50%低于玩家，30%相同，20%高于玩家

**仇人来源：**
- 强抢、出卖同门、副本杀人
- 争夺资源时对方逃跑后复仇
- 随机事件中无意得罪

## 十二、宗门系统

### 三条路线
| | 正道宗门 | 魔道宗门 | 散修 |
|--|---------|---------|------|
| 资源 | 稳定中等 | 丰厚 | 少但机缘大 |
| 寿命 | 正常 | 上限-15% | 正常但消耗更多 |
| 负面情绪 | 积累慢 | 积累快 | 看选择 |
| 仇家 | 少 | 多 | 中等 |
| 战斗 | 普通 | 普通 | 每回合+1胜负 |
| 冲关 | 稳健 | 心魔风险大 | 基础略低但机缘加成 |
| 限制 | 多 | 中等 | 无 |
| 特殊机缘 | 少 | 无 | 多 |

### 路线选择
- 通过早期事件自然引导，多次询问，不强制
- 路线可中途改变，代价极大

### 路线切换代价
**正道宗门 → 背叛：**
- 仇人池+2，戾气+30，执念+20
- 宗门专属道具失效，冲关-10%持续2境界
- 其他正道宗门永久关闭

**魔道宗门 → 脱离：**
- 仇人池+3，负面情绪不消除
- 解除寿命-15%限制

**散修 → 宗门：**
- 代价最小，散修战斗加成消失，散修专属机缘关闭

### 宗门任务
- 必然触发战斗
- 奖励固定丰厚，不能拒绝
- 类型：讨伐妖兽、护送长老、清剿魔修、争夺灵脉

## 十三、功法系统

- 正道/魔道功法与宗门绑定，散修天生修炼无需功法
- 功法平时无用，加入宗门时才能修炼
- 必须持有对应功法才能加入对应宗门

### 功法获取概率
| 功法类型 | 事件掉落 | 战斗获得 | 宗门赠与 |
|---------|---------|---------|---------|
| 正道功法 | 10% | 15% | 25% |
| 魔道功法 | 5% | 8% | 10% |

### 功法特点
**正道功法：**
- 修炼速度稳定（基准×1.0）
- 根基扎实，冲关+5%隐性加成
- 负面情绪积累慢

**魔道功法：**
- 修炼速度更快（每事件修为×1.5）
- 根基不稳，冲关-5%隐性减成
- 负面情绪积累更快
- 寿命上限-15%，低境界省寿命，高境界压力大

**散修：**
- 修炼速度中等（基准×1.0）
- 无根基加减成
- 有概率触发奇遇功法（极稀有，效果远超正魔）

## 十四、受伤状态系统

### 受伤分级
| 等级 | 触发 | 影响 |
|------|------|------|
| 轻伤 | 战斗失败/普通副本 | 修炼-20%，闭关-20% |
| 重伤 | 危险副本/强敌战败 | 修炼-50%，闭关-50%，无法进副本 |
| 濒死 | 极危险情况 | 修炼-80%，无法做任何主动事件 |

叠加规则：轻伤未治再受伤→重伤，重伤未治再受伤→濒死触发续命丹。

### 消除方式
| 方式 | 消耗 | 可治等级 |
|------|------|---------|
| 养伤丹 | 150灵石 | 全部 |
| 医修治疗 | 轻伤100/重伤300/濒死800灵石 | 全部 |
| 休养（轻伤） | 见下表 | 仅轻伤 |
| 休养（重伤） | 见下表 | 仅重伤 |

**休养寿命消耗：**
| 境界 | 轻伤 | 重伤 |
|------|------|------|
| 炼气期 | 5年 | 15年 |
| 筑基期 | 12年 | 35年 |
| 结丹期 | 30年 | 90年 |
| 元婴期 | 70年 | 200年 |

## 十五、飞升结局

- 化神期才有资格触发飞升
- 玩家主动选择，不飞升则耗尽寿命自然坐化
- 纯概率事件，玩家无法干预，结果展示一段文字

### 天命值
- 全程隐藏，飞升时揭晓，初始值50，范围0-100

**加天命值：**
| 行为 | 加值 |
|------|------|
| 救助受伤修士 | +3 |
| 放弃资源成全他人 | +5 |
| 帮助弱者对抗强敌 | +8 |
| 完成因果报恩 | +5 |
| 放走可以击杀的仇人 | +6 |
| 渡人突破境界 | +10 |
| 守护无辜者免受伤害 | +7 |

**减天命值：**
| 行为 | 减值 |
|------|------|
| 背叛宗门/同门 | -10 |
| 滥杀无辜 | -8 |
| 强抢他人资源 | -6 |
| 走捷径损根基 | -4 |
| 出卖他人换取利益 | -8 |
| 修炼魔功 | -5 |
| 临阵逃跑（每次） | -2 |

### 飞升成功率
| 天命值 | 飞升成功率 |
|-------|----------|
| 0-20 | 10% |
| 21-40 | 25% |
| 41-60 | 45% |
| 61-80 | 65% |
| 81-100 | 85% |

### 结局文字
| 结局 | 基调 | 示例 |
|------|------|------|
| 飞升成功 | 壮阔 | 「踏破虚空，你终于飞升灵界……」 |
| 自然坐化 | 平静 | 「你安然坐化，留下一缕残魂，融入天地之间……」 |
| 冲关陨落 | 悲壮 | 「天雷滚滚，你在渡劫中燃尽最后一丝神识……」 |
| 心魔陨落 | 悲剧 | 「心魔终究吞噬了你，你在自己的执念中永远沉沦……」 |
| 战斗陨落 | 壮烈 | 「你倒在了修仙路上，鲜血染红了脚下的土地……」 |
| 飞升失败 | 遗憾 | 「距离灵界只差一步，你却永远停在了这里……」 |

## 十六、全局字段总览

| 字段 | 可见性 | 作用 |
|------|--------|------|
| 修为 | 可见（血条） | 当前境界进度，决定冲关资格 |
| 寿命 | 可见（血条） | 时间压力，耗尽坐化 |
| 向道之心 | 可见 | 影响冲关成功率 |
| 情绪值（正/负） | 部分可见 | 仅影响结丹→元婴心魔关 |
| 天命值 | 隐藏 | 影响最终飞升概率，飞升时揭晓 |
| 根基状态 | 隐性 | 影响冲关，走捷径会受损 |
| 仇人池 | 可见 | 待解决的仇人列表 |
| 经验值 | 可见 | 冲关失败积累，上限+15% |
| 受伤状态 | 可见 | 有伤时显示，影响修炼和行动 |
